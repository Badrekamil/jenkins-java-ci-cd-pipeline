# Jenkins CI/CD Pipeline for Java Application

## Description
Implemented a CI/CD pipeline using Jenkins for a Java Maven application by integrating an external GitHub repository.

## Tech Stack
- Jenkins
- Maven
- GitHub
- Linux

## Features
- Integrated Jenkins with external GitHub repository
- Automated build and test process using Maven
- Implemented pipeline using Jenkinsfile
- Automated artifact archiving

## Pipeline Workflow
1. Jenkins pulls code from GitHub repository
2. Builds project using Maven
3. Runs unit tests
4. Archives build artifacts
